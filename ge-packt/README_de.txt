Dieses Verzeichnis enthält alle Shell- und vim-Skripte, die im Buch "vim
ge-packt" (mitp Verlag Bonn 2004, ISBN 3-8266-1425-9) mit einem Hinweis auf die
Webseiten home.wtal.de/rwobst/vim/, www.mitp.de und www.vim.org enthalten sind.
Im einzelnen sind das:

My_input.vim	- Debugging-Hilfsfunktion, vgl. 5.8
center.vim	- Skript zum Zentrieren von Text (mit ":so" zu laden)
colortest.vim	- Testen der Colorschemes, vgl. 6.9.2
crypt.vim	- Editieren GnuPG-chiffrierter Dateien, vgl. 7.3
emailfold.vim	- Faltung von zitierter Mail, vgl. 6.5.2
mkfliess.vim	- Erzeugen von Fließtext, vgl. 5.7
myconfig	- Flags zur Compilierung von vim mit Python-Unterstützung
myvimspell.vim	- Plugin zum Spellcheck, vgl. 7.6
sywdh.vim	- Häufig wiederholte Wörter einfärben, vgl. 7.5.1
vic		- Shellskript zum Editieren von C-Programmen, vgl. 7.9
vic.vim		- zu vic gehöriges Plugin, vgl. 7.9
vicc		- analoges Shellskript zu vic, aber für C++
vil		- Shellskript: vim als "less" mit Syntaxfarben, vgl. 7.8
vimrc		- Meine Datei .vimrc, vgl. 4.5
vimspell.vim	- Das spell-Plugin von Clabaut et al., vgl. 7.6
visp		- Shellskript zur Verwendung von vimspell.vim
viw		- Shellskript zum Text formatieren mit vim, vgl. 7.2
vixml		- Shellskript zum Editieren von XML-Dateien, vgl. 7.4
wdh		- Muster-Datei für wdh.vim
wdh.vim		- Plugin für Stilverbesserung, vgl. 7.5.2
wipeout		- Shellskript zum Überschreiben einer Datei (für crypt.vim)
wordcmp.c	- Mein Programm zum Textvergleich (Doku enthalten), vgl. 6.12
xml.vim		- Plugin zum XML-Editieren, vgl. 7.4

Außerdem ist darin enthalten:

gpg.vim		- Entpacken und Decodieren chiffrierter Mail, vgl. o.g.

quopriconv	- dazu benötigtes Python-Skript
