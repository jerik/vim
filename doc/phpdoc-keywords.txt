*phpdoc-keywords*	Hilfe um schnell die phpDocumentator Schlüsselwörter
					nachzusclagen.

Kurzreferenz der Parameter:							*phpdoc-toc*

	|phpdoc-param|		@param 		string $sString
	|phpdoc-return|		@return 	integer $iReturn
	|phpdoc-uses|		@uses		foo::bar
	|phpdoc-todo|		@todo		refactoring machen
	|phpdoc-copyright|	@copyright	Copyright © 2005
	|phpdoc-version|	@version 	1.0.9
	|phpdoc-exmaple|	@exmaple	/tmp/file.php
	|phpdoc-package|	@package	MyPackage

Beschreibung:										*phpdoc-description*	
	Die ersten Zeilen nach dem Einleitenden Kommentrazeichen '/**' werden 
	als Beschreibung der Funktion/Klasse/etc benutzt. Dabei müssen diese 
	durch eine leere kommentarzeile von den @Attributen getrennt sein:

		/**
		* Die ersten Zeilen bis zur leerzeile werden
		*   als Beschreibung interpretiert
		* 
		* @author   J. Erik Heinz
		*/

@Attributes:										*phpdoc-attributes*


  @param:											*phpdoc-param*
	Angabe der parameter einer Funktion. Der Tag kann mehrmals vorkommen, je 
	nachdem wieviele Parameter die Funktion hat Dabei werden der type des 
	Parameters und optional eine Beschreibung angegeben

		* @param string $sId    String variable
		* @param boolean $bTest 
		* @param array $aArray  Hold the parameter for the function

  @return:											*phpdoc-return*
	Angabe des Rückgabewertes. Wenn kein type angegeben wird, wird es 
	defaultmässig als 'mixed' angesehn
	
    	* @return   true

  @uses:											*phpdoc-uses*	
	erzeugt eine Referenz auf eine Funtion die benutzt wird. Gleichzeitg wird 
	von der benutzen Funktion eine Rück-Referenz auf die benutzenden Funktion 
	gemacht Gilt auch für extern Funktionen, als auch Variablen
	
	    * @uses myfunc
    	* @uses sVariable
		* @uses Anotherclass::otherfunc

  @todo:											*phpdoc-todo*
	Hier können todo-sachen hereingeschrieben werden. Diese tauchen dann 
	neben der funktion auch aggregiert in einer seperaten Lite auf.

    	* @todo vim helppage zu den keywords für phpdocumentator machen

  @copyright:										*phpdoc-copyright*
	Hier wird das Copywright spezifiziert.

    	* @copyright    Copyright © 2005, J. Erik Heinz

  @version:											*phpdoc-version*	
	Angabe der Version

		* @version  0.2.1

  @example:											*phpdoc-example*
	Bindet eine Beispieldatei ein. Diese Datei wird mit highlight_file geparst 
	und in die Dokumentation eingebunden. Das Attribut überprüft ob die Datei
	existiert.  Man kann eine relative oder absoluten Pfad angeben. rtfm
	
    	* @example /tmp/exmaplefile.php

  @package											*phpdoc-package*
	Damit kann man die Datei einem package zuordnen.  Wird aber nicht von allen 
	Konvertern berücksichtigt

    	* @package  MyPackage
